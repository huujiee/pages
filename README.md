# Eyayoo Studio – Public Pages Mirror

This repository hosts the built static site for public hosting via GitHub Pages (site branch: `publish`). It is a deployment mirror only — the source code lives in a private repository.

- Live site: https://huujiee.github.io/pages/
- Optional custom domain: if configured, this mirror may also be reachable at your custom domain (see CNAME).

Usage
- Do not open pull requests here. Files are generated and pushed by CI from the private repo.
- Issues about site content should be filed in the main/private repository.
